博客搭建那些坑

 by Aeolia on 2015年4月21日

前段时间搭建完这个博客，中间踩了一些小坑，稍微记录一下。

  1. github pages出不来。
    按照github pages的文档将内容上传到gh-pages分支后，该目录下的index.html并没有被github的服务器解释出来。我这里的错误是该目录下_config.yml的语法有错，_config文件中的键值对，冒号后面应该有一个空格。具体其他错误，可以到对应仓库的settings里的github pages那一项看报错情况。
  2. hexo-less编译报错:one ERROR about asset render failed
    解决方法参见issue;
  3. 使用hexo的deploy不成功。
    windows下不能使用命令行窗口执行这一条指令，而需使用git bash
  4. jade报错。
    空格和缩进不能混用。
